Clipboard

Clipboard was a project I undertook in CS350, a class that taught the fundamentals of Software Engineering.

In this class each group had four members with a role for each member.

My group's task was to create a project-tracking taskboard.

The roles in this class were Design, Requirements, Code, and Test, rotating roles each milestone.

For this project, I was the Test role for the first milestone, and rotated through the roles in the order I mentioned them.

There were 7 milestones in this project, each representing important steps in the Software Engineering process.

We kept track of our progress on github in .md documents, and you can click through each milestone below to see a detailed view
